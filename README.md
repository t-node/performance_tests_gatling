Performance Testing using Gatling
=========================

Get the project
---------------

```bash
git clone https://github.com/saksdirect/performance-testing-gatling.git
```

Start SBT
---------
```bash
$ sbt
```

Run all simulations
-------------------

```bash
> test
```

Run a single simulation
-----------------------

```bash
> testOnly "testClass"
```

List all tasks
--------------------

```bash
> tasks
```
